JSBJJ Student Analytics Dashboard

A Python-based data analysis tool designed to monitor student attendance and engagement trends within a Brazilian Jiu-Jitsu academy.

📌 Overview 
This project processes structured CSV datasets to: 
Compare active and frozen membership records Identify inactive or low-engagement members Analyse participation patterns over time Support operational decision-making with structured data insights

🛠 Technologies Used 
Python CSV data processing 
Data filtering and aggregation logic 
Basic data comparison techniques

📂 Project Structure jsbjj/ │
├── app.py
├── requirements.txt

⚙ How It Works 
Loads structured CSV datasets Processes and compares membership records 
Identifies inactive members Generates structured outputs for operational review

🚀 Purpose
This project was created to apply structured problem-solving and data analysis concepts to a real-world operational scenario, improving retention visibility and engagement tracking.

🔒 Note 
Sample or anonymised data should be used when sharing publicly.
